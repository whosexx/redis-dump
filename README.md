# redis-dump

dump redis key or restore redis key

Usage of redis-dump
  -P string
        password
  -h string
        host (default "localhost")
  -k string
        key fiter (default "*")
  -m string
        support dump or restore (default "dump")
  -n int
        redis database index
  -o string
        dump or restore file path (default "redis.dump")
  -p int
        redis port (default 6379)
